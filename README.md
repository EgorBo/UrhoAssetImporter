## UrhoAssetImporter - a simple UrhoSharp assets viewer + AssetImporter

Supported formats:
- .obj, .dae, .fbx, .blend, .3ds, .lwo, [etc](http://assimp.sourceforge.net/main_features_formats.html). 
- Urho3D Materials, Particles, Prefabs, Models, etc

Supported platforms:
- Windows (WPF)
- macOS (Cocoa)
